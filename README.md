# 📄 Weebs Docs

Sistema desktop em Python para geração automática de documentos profissionais em PDF.

Permite criar contratos, orçamentos, recibos, propostas comerciais, ordens de serviço e termos de confidencialidade de forma rápida, organizada e com interface moderna.

---

## 🚀 Funcionalidades

- 📑 Geração de documentos em PDF com layout profissional  
- 💰 Orçamentos automáticos  
- 🧾 Recibos de pagamento  
- 📈 Propostas comerciais  
- 🛠️ Ordens de serviço  
- 🔒 Termos de confidencialidade (NDA)  
- 🧠 Formulários dinâmicos por tipo de documento  
- 🎨 Interface moderna com tema escuro  
- 💾 Salvamento direto do PDF no computador  
- ⚡ Fluxo rápido e intuitivo  

---

## 🖥️ Interface

O sistema conta com uma interface desktop moderna baseada em Tkinter:

- Tema escuro (dark UI)
- Cards interativos para seleção de documentos
- Formulários dinâmicos
- Feedback visual e mensagens de sucesso/erro
- Ícone personalizado da Weebs 💜

---

## 📦 Tecnologias utilizadas

- Python 3
- Tkinter (GUI)
- FPDF2 (geração de PDF)
- Base64 (ícone embutido)
- Datetime

---

## ▶️ Como executar

    pip install fpdf2
    python weebs_docs.py

---

## 📁 Estrutura do projeto

    weebs_docs.py

Projeto totalmente single-file para facilitar execução e portabilidade.

---

## 💡 Como funciona

1. Escolha o tipo de documento  
2. Preencha os campos solicitados  
3. Clique em “Gerar PDF”  
4. Escolha onde salvar  
5. Documento gerado automaticamente
